# LMS - Frontend

### Set up Instruction

1. Clone the project
'''
 git clone - https://github.com/ravikumar50/LMS.git
'''

2. move to the Directry
'''
  cd LMS-FrontEnd
'''

3. Install dependencies
'''
    nmp install
'''

4. Run the server
'''
  npm run dev
'''


Install Tailwind CSS
